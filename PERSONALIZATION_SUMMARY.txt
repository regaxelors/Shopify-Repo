FEATURES:
────────────────────────────────────────────────────────────────────────────────

1. TEXT PERSONALIZATION
   - Custom text input with character limits (configurable)
   - Font selection, color, size options
   - Placeholder text for guidance

2. IMAGE PERSONALIZATION
   - Custom image upload (JPG, PNG, GIF)
   - File size limits (default 10MB)
   - Positioned front or back

3. COMBINED (TEXT + IMAGE)
   - Single product with both custom text and image
   - Example: Team jersey (name + logo)

4. SHOPIFY INTEGRATION
   - Auto-adds tags: "personalizable", "custom-text", "custom-image"
   - Enables Shopify's live preview UI
   - Buyers see real-time customization preview

5. CONTENT REVIEW WORKFLOW
   - PERSONALIZATION_HOLD_FOR_REVIEW=true (default)
     → Products created as "draft"
     → Admin reviews before auto-publishing
     → Prevents offensive text reaching production
   - PERSONALIZATION_HOLD_FOR_REVIEW=false
     → Auto-publishes immediately (less safe)

USAGE:
────────────────────────────────────────────────────────────────────────────────

// Custom Name Poster
const result = await printifyService.runPipeline({
  title: 'Custom Name Poster',
  personalizationType: 'text',  // Enable text personalization
  dryRun: false,
});

// Result:
// - Product created with "Enter your name" text layer
// - Tags: [original tags + "personalizable", "custom-text"]
// - Status: "draft" (held for admin review)
// - Shopify recognizes "personalizable" tag
// - Admin reviews and publishes

WORKFLOW:
────────────────────────────────────────────────────────────────────────────────

1. runPipeline(personalizationType: 'text')
2. buildPersonalizationPayload() creates layers
3. Add tags: "personalizable", "custom-text"
4. Set status: "draft" (for review)
5. Admin reviews in Printify
6. Admin publishes to Shopify
7. Shopify renders live preview UI
8. Customer enters custom text
9. Order sent to print with customization
10. Product printed and shipped
